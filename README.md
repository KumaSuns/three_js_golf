# たっくんとGOLF

ブラウザで遊べる Three.js 製のミニゴルフゲームです。

## 遊び方

1. タイトル画面で **スタート**
2. **① スタート** → パワーゲージが動く
3. **② パワー確定** → クラブを振り切る
4. **③ インパクト**（緑ゾーンで Space）→ ショット
5. 3 ホールを回ってスコアを競う

### 操作

| 操作 | キー / UI |
|------|-----------|
| ショット進行 | `Space` / `Z` / `Enter`、画面下ボタン |
| クラブ変更 | `Q` / `E`、クラブ選択ボタン |
| 打ち方向 | `←` `→`、方向パッド |
| カメラ | `↑` `↓`、カメラパッド |
| 飛距離表示後 | `Space` で次へ / `R` で打ち直し |

### パター（PT）

インパクトはありません。**パワー確定でそのまま打ちます。**

## 起動方法

PHP が使える環境でプロジェクトルートから:

```bash
php -S localhost:8080
```

ブラウザで `http://localhost:8080/index.php` を開いてください。

静的ホストのみの場合は `index.php` を `index.html` にリネームしても動作します（PHP コードは含まれていません）。

## 技術

- [Three.js](https://threejs.org/) r128（CDN）
- Vanilla JavaScript / CSS
- 3 ホール（PAR 3 / 4 / 5）、7 クラブ
